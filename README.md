# 10 keys - a keyboard with *drumrole* 10 KEYS!!!

To run this project, you need:
1. Arduino Mega (or any other Arduino)
2. 10 keys (duh)
3. Brownie points for "shift"ing

# To run:
### 1. Upload the ino sketch to the microcontroller
### 2. Run python script to convert serial input to key strokes
- Install the required libraries
```
pip install pyserial pynput
```
- Run the script
```
python keystroke.py
```
